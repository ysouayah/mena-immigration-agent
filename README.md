# MENA Advocacy & Policy Agent
An AI-powered dashboard for analyzing migration data and U.S. immigration policy in the MENA region.

## Setup Instructions
1. **Clone the repo.**
2. **Install dependencies:** `pip install -r requirements.txt`
3. **Set up Environment Variables:**
   - Create a `.env` file in the root directory.
   - Add your database connection string: `DATABASE_URL=your_postgresql_url_here`
4. **Run the App:** `streamlit run app.py`

## Features
- Interactive displacement map using UNHCR data.
- RAG-powered AI agent for policy analysis.
