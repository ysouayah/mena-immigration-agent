# MENA Advocacy & Policy Agent
An AI-powered dashboard for analyzing migration data and U.S. immigration policy in the MENA region, as well as a built-in agent that outputs up-to-date information regarding US immigration policies with respect to individual MENA countries.

## Setup Instructions
1. **Clone the repo.**
2. Add your database connection string in every .py file: `DATABASE_URL=your_postgresql_url_here`
3. **Run the App:** `streamlit run app.py` in the terminal

## Features
- Interactive displacement map using UNHCR data.
- RAG-powered AI agent for policy analysis.
